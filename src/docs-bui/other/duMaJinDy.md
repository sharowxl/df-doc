### 属性
| 参数      | 说明    | 类型      | 可选值       | 默认值   |
|---------- |-------- |---------- |-------------  |-------- |
| requireDuMaLyr | 是否必填领药人 | boolean | - | false |
| appInfo | 应用信息 | object | - | - |
| keShiList | 字典科室list | array | - | - |

### Methods

| 方法名 | 说明 | 参数 | 参数说明|
| ---- | ---- | ---- |----|
| open | 打开毒麻精打印窗口，返回promise，等待打印完成返回true，关闭弹窗返回false | yiZhuList: Array, bingRenZyId: String | - |
