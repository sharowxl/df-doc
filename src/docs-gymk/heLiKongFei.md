## 合理控费模块

### 在页面中使用
通过this.$heLiKongFei调用，等价于window.yeWuGyMk.heLiKongFei.action，test.json文件内容见最后

```html
<template>
  <df-button
    :loading="loading"
    @click="handleClick"
  >
    测试
  </df-button>
</template>
<script>
import data from './test.json'

export default {
  name: 'Demo',
  data () {
    return {
      loading: false
    }
  },
  methods: {
    handleClick () {
      this.loading = true
      this.$heLiKongFei(data).then(res => {
        this.loading = false
        console.log('rr', res)
      }).catch(e => {
        console.error('helikongfei', e)
        this.loading = false
      })
    }
  }
}
</script>
```
